# 吞嚥困難篩檢問卷 Dysphagia Screening Questionnaire

一個基於 **Ohkuma R, et al. (2002)** 吞嚥困難篩檢問卷的互動式網頁版。

## 功能

- 📋 完整 15 題吞嚥困難篩檢問卷
- ✅ 即時進度顯示（已填答 / 總題數）
- 📊 自動評分：依據選項 A / B / C 判定結果
- 🖨️ 友善列印格式
- 📱 RWD 行動裝置適用

## 評分方式

| 結果 | 條件 | 建議 |
|------|------|------|
| ⚠️ 吞嚥困難 | 任一題選 **A**（經常／明顯反覆） | 建議專業吞嚥評估 |
| 🔍 疑似吞嚥困難 | 僅選 **B**（有時候／很少），無 A | 建議觀察或諮詢 |
| ✅ 無明顯異常 | 全部選 **C**（沒有） | 維持良好習慣 |
